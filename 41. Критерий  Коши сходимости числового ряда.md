## Теорема

$$ \Large
\text{Для того, чтобы ряд $\sum_{n=1}^{\infty} a_n$ сходился, }
$$
$$\Large
\text{необходимо и достаточно, чтобы } \forall \varepsilon > 0 \ \exists N
$$

$$ \Large
\text{т.ч. } \forall m > n > N \ \text{выполнялось}
$$

$$ \Large
\left| \sum_{k=n+1}^{m} a_k \right| < \varepsilon
$$
---
### Доказательство

$$ \Large
\text{По критерию Коши существования конечного предела последовательности(1.11),}
$$

$$ \Large
\text{для того, чтобы последовательность } \{S_n\}_{n=1}^{\infty} \text{ имела конечный предел, }
$$

$$ \Large
\text{необходимо и достаточно, чтобы } \forall \varepsilon > 0 \ \exists N \ \text{т.ч. } \forall m > n > N \ \text{выполнялось}
$$

$$ \Large
|S_m - S_n| < \varepsilon
$$

$$ \Large
\text{Поскольку } S_m - S_n = a_{n+1} + \dots + a_m = \sum_{k=n+1}^{m} a_k \  \Rightarrow \left| \sum_{k=n+1}^{m} a_k \right| < \varepsilon
$$
