## Определение
$$ \Large 
\text{Пусть } \{f_n(x)\}_{n=1}^\infty; \quad f: X \to \mathbb{R}
$$ 
$$ \Large 
\text{говорят, что } \{f_n(x)\} \text{ равномерно сходится в } f(x) \text{ при } n \to \infty 
$$
$$\Large
\text{по другому записывается:} f_n(x) \underset{\substack{n \to \infty \\ x \in X}}{\rightrightarrows} f(x)
$$
$$ \Large 
\text{если } \forall \varepsilon > 0 \ \exists N \text{ т.ч. } \forall n > N \text{ и } \forall x \in X \text{ выполняется: }  
$$
$$ \Large
|f_n(x) - f(x)| < \varepsilon 
$$

## Отличие равномерной от поточечной сходимости

крч основная залупа вот в чем: равномерная сходимость = сходится начиная с какого-то N при любом x, а поточечная = сходится также при любом x, но для каждого x существует свое N которые могут быть разными
Следовательно равномерная сходимость является поточечной, но поточечная не является равномерной.
