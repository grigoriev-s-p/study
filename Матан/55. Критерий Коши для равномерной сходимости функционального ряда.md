## Теорема
$$ \Large
\text{Для того чтобы ряд } \sum_{n=1}^\infty v_n(x) \text{ равномерно сходился на } X 
$$ 
$$\Large
\text{необходимо и достаточно, чтобы: }
$$
$$ \Large
\forall \varepsilon > 0 \quad \exists N \text{ т.ч. } \forall m > n > N \text{ и } \forall x \in X
$$
$$ \Large
\text{вып. } |v_{n+1}(x) + \dots + v_m(x)| < \varepsilon 
$$ 
## Доказательство 
$$ \Large 
 \text{ По определению равномерно сходящегося ряда: }
$$
$$ \Large
S_k(x) = v_1(x) + \dots + v_k(x) \text{ равномерно сходится на } X \Rightarrow
$$
$$ \Large
\forall \varepsilon > 0 \quad \exists N \text{ т.ч. } \forall m > n > N \text{ и } \forall x \in X \text{ вып. } |S_m(x) - S_n(x)| < \varepsilon 
$$ 
$$ \Large 
\text{так как } S_m(x) - S_n(x) = v_{n+1}(x) + \dots + v_m(x) \Rightarrow 
$$
$$ \Large
 |S_m(x) - S_n(x)| < \varepsilon  \iff |v_{n+1}(x) + \dots + v_m(x)| < \varepsilon
 $$
