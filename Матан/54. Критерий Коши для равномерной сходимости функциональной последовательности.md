## Теорема
$$ \Large
\text{Пусть} \ {f_n(x)}_{n=1}^\infty \text{}
$$ 
$$ 
\Large \text{Для того чтобы } \exists f: X \to \mathbb{R} \ \text{ т.ч. } \ f_n(x) \underset{\substack{n \to \infty \\ x \in X}}{\rightrightarrows} f(x)
$$
$$ \Large
\text{необходимо и достаточно, чтобы } \forall \varepsilon > 0 \ \exists N \text{ т.ч. } \forall n, m > N \text{ и } \forall x \in X \text{}
$$ 
$$ \Large 
\text{выполнялось } |f_m(x) - f_n(x)| < \varepsilon 
$$
### Доказательство
1) =>
$$ \Large
\text{Пусть} \ \exists f \text{такая, к которой последовательность сходится} \Rightarrow
$$ 
$$ \Large 
f_n(x) \underset{\substack{n \to \infty \\ x \in X}}{\rightrightarrows} f(x)
\Rightarrow \forall \varepsilon > 0 \ \exists N \text{ т.ч. } \forall n > N \text{ и } \forall x \in X 
$$
$$\Large
\text{ выполнено } |f_n(x) - f(x)| < \frac{\varepsilon}{2} 
$$

$$ \Large
\text{Пусть } m > N, \ n > N \quad \forall x \in X:
$$
$$ \Large |f_m(x) - f_n(x)| = |(f_m(x) - f(x)) + (f(x) - f_n(x))| \le$$

$$ \Large
\le |f_m(x) - f(x)| + |f(x) - f_n(x)| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon
$$
2) <=
$$ \Large
\Leftarrow \text{ Пусть выполнено }|f_m(x) - f_n(x)| < \varepsilon 
$$
$$\Large
\text{ Ищем нужную функцию}
$$ 
$$ \Large 
\text{Фиксируем } x \in X; \text{ для } \{f_n(x)\} \text{ (числовой последовательности) }
$$
$$\Large
\text{выполнен Критерий Коши, т.е.}
$$
$$ \Large
|f_m(x) - f_n(x)| < \varepsilon \Rightarrow \exists \lim_{n \to \infty} f_n(x) \in \mathbb{R}
$$ 
$$ \Large 
\text{Когда меняем } x, \text{ получаем другое значение - Пусть }\lim_{n \to \infty} f_n(x) = f(x)
$$
$$ \Large
\text{Выясним, что равномерно сходится именно к } f(x); 
$$
$$ \Large 
\forall \varepsilon > 0 \quad |f_m(x) - f_n(x)| < \varepsilon \quad m > n > N; \text{ фиксируем } x
$$

$$ \Large
\Rightarrow \lim_{m \to \infty} |f_m(x) - f_n(x)| \le \varepsilon \text{ (при переходе к пределу получаем нестрогое нер-во)}
$$ 
$$ \Large 
 \Rightarrow |f(x) - f_n(x)| \le \varepsilon \quad \forall x \in X
$$
$$ \Large
\text{это верно при } \forall n > N \text{ и } \forall x \in X \quad \text{ч.т.д. } 
$$
